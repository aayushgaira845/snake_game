# 🐍 Snake Game (Java)

A classic **Snake Game** built using **Java and Swing**.  
This project recreates the traditional snake gameplay where the player controls a snake that grows in size as it eats apples. The objective is to achieve the highest score without colliding with the walls or the snake's own body.

---

## 📌 Project Overview

This Snake Game is a desktop application developed in **Java** using the **Swing GUI framework**.  
The game demonstrates fundamental programming concepts such as:

- Object Oriented Programming (OOP)
- Event handling
- GUI development
- Game loop logic
- Collision detection

The player controls the snake using keyboard arrow keys. The snake grows longer each time it eats an apple, increasing the challenge of avoiding collisions.

## 🎮 How to Play

1. Run the program.
2. Use the **Arrow Keys** to control the snake:

| Key | Action |
|----|------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

3. Eat apples to increase your score.
4. Avoid:
   - Hitting the walls
   - Hitting your own snake body

If a collision happens, the game ends and **Game Over** is displayed.
